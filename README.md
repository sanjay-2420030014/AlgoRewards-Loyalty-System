# AlgoRewards Loyalty System

## Objective
This project aims to leverage the Algorand Blockchain to create a transparent, low-cost loyalty reward system. Businesses can mint "Reward Tokens" and distribute them to customers instantly.

## Why Algorand?
- **Speed:** Transactions are finalized in ~2.8 seconds.
- **Low Cost:** Each reward distribution costs only 0.001 ALGO.
- **ASAs:** Using Algorand Standard Assets makes token creation simple and secure.

## Project Structure
- `/contracts`: Future smart contract logic for automated rewards.
- `/scripts`: Python scripts for minting and transferring tokens.
- `README.md`: Project documentation.
